This driver works*

Driver written for MTK3339 GPS module. Requires files generated
by PSoC Creator for UART operation.

***CREATE AND NAME UART BLOCK "GPS_UART" IN THE TOP DESIGN FILE***

***CREATE AND NAME CLOCK "UART_CLOCK" IN TOP DESIGN FILE***

***SET CLOCK TO 48MHZ***

***SET BAUDRATE TO 9600***
